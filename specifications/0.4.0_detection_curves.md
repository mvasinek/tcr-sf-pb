CHANGELOG

Doplň:

## [0.4.0] - Detection curve summaries and plots

### Added
- Construction of detection curve point table.
- Empirical detection summaries for SF-to-blood and blood-to-SF directions.
- Default clone-size bins.
- PNG plots for detection curves.
- CLI for detection curve analysis.
- Optional filters for cell type and minimum source clone size.
Kritéria dokončení

Feature je hotová, pokud:

lze načíst paired_detection_table.csv,
vznikne detection_curve_points.csv,
vznikne detection_curve_summary.csv,
správně se počítají oba směry detekce,
správně se přiřazují biny velikosti klonu,
správně se počítá empirická pravděpodobnost detekce,
vzniknou dva PNG grafy, pokud není použit --no-plots,
fungují parametry --cell-type, --min-source-cells, --no-plots,
všechny testy projdou přes pytest,
projekt má verzi 0.4.0.
