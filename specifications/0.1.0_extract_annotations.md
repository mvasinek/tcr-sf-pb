# Feature 0.1.0 – Extraction of TCR annotation data

## Cíl
Vytvořit první verzi knihovny, která projde vstupní adresář, najde všechny soubory končící suffixem:

```text
annotations.csv.gz

rozbalí je, načte jejich obsah, vyextrahuje potřebná metadata a vytvoří jeden sjednocený výstupní CSV soubor pro další analýzu klonů mezi krví a synoviální tekutinou.

Verze

Nastav verzi projektu na:

0.1.0

Použij sémantické verzování.

Vstup

Adresář obsahující soubory typu například:

GSM4859841_PM1_CD4_SF_p7_filtered_contig_annotations.csv.gz
GSM4859842_PM4_CD4_blood_p1_filtered_contig_annotations.csv.gz

Agent musí rekurzivně najít všechny soubory odpovídající:

*annotations.csv.gz
Metadata extrahovaná z názvu souboru

Z názvu souboru vyčti:

Pole	Příklad
gsm_id	GSM4859841
sample_group	PM1
cell_type	CD4
compartment	SF nebo blood
patient	p7

Příklad:

GSM4859841_PM1_CD4_SF_p7_filtered_contig_annotations.csv.gz

má dát:

gsm_id = GSM4859841
sample_group = PM1
cell_type = CD4
compartment = SF
patient = p7
Sloupce načtené z annotation souborů

Ze souboru zachovej minimálně tyto sloupce:

barcode
is_cell
contig_id
high_confidence
length
chain
v_gene
d_gene
j_gene
c_gene
full_length
productive
cdr3
cdr3_nt
reads
umis
raw_clonotype_id
raw_consensus_id

Ke každému řádku přidej metadata z názvu souboru:

source_file
gsm_id
sample_group
cell_type
compartment
patient
Filtrování

Ve verzi 0.1.0 zatím data pouze načti a sjednoť.

Nepouštěj zatím žádné agresivní biologické filtrování.

Přidej ale volitelný parametr:

--productive-only

Pokud je použit, ponech pouze řádky splňující:

is_cell == True
high_confidence == True
full_length == True
productive == True
chain in ["TRA", "TRB"]
cdr3 not null
Výstup

Vytvoř jeden výstupní soubor:

combined_annotations.csv

Výstupní tabulka musí mít tento tvar:

source_file,gsm_id,sample_group,cell_type,compartment,patient,barcode,is_cell,contig_id,high_confidence,length,chain,v_gene,d_gene,j_gene,c_gene,full_length,productive,cdr3,cdr3_nt,reads,umis,raw_clonotype_id,raw_consensus_id
CLI rozhraní

Vytvoř jednoduchý příkaz:

python -m tcr_bcr_tools.extract_annotations --input-dir ./data --output ./outputs/combined_annotations.csv

S volitelným filtrem:

python -m tcr_bcr_tools.extract_annotations --input-dir ./data --output ./outputs/combined_annotations.csv --productive-only

Založ CHANGELOG.md:

# Changelog

## [0.1.0] - Initial annotation extraction

### Added
- Recursive discovery of `annotations.csv.gz` files.
- Parsing of GEO/sample metadata from filenames.
- Combination of contig annotation files into one CSV table.
- Optional productive-only filtering.
- Basic unit tests.
Kritéria dokončení

Feature je hotová, pokud:

lze spustit CLI příkaz nad adresářem s .csv.gz soubory,
vznikne jeden combined_annotations.csv,
každý řádek obsahuje metadata ze souboru i původní annotation sloupce,
testy projdou přes pytest,
verze projektu je 0.1.0.
