# Feature 0.5.5 – Results Browser & Interactive Visualization

# Cíl

Rozšířit verzi **0.5.4** o plnohodnotný prohlížeč výsledků analýz.

Od této verze se aplikace stává interaktivním analytickým prostředím, ve kterém uživatel nemusí hledat výsledky v adresářové struktuře, ale všechny výstupy jsou automaticky registrovány, indexovány a vizualizovány.

GUI se stává primárním způsobem práce s projektem.

Veškerá analytická logika zůstává v knihovně.

GUI pouze načítá výsledky z Output Registry.

# Kritéria dokončení

Feature je dokončena pokud:

- existuje Output Registry API,
- GUI používá Output Registry místo filesystemu,
- funguje Results Browser,
- funguje CSV Viewer,
- funguje Figure Gallery,
- funguje Table Gallery,
- funguje Search,
- funguje Favorites,
- funguje Recent,
- funguje Metadata panel,
- funguje Output comparison,
- funguje ZIP export,
- existuje Project Summary dashboard,
- Git metadata jsou součástí každého výstupu,
- GitHub Actions procházejí,
- všechny testy procházejí,
- verze projektu je **0.5.5**,
- po dokončení bude vytvořen Git tag:

```
v0.5.5
```

# Poznámka pro budoucí vývoj

```
0.5.6 – Report Generator (HTML/PDF, publikovatelné reporty)
```

## Architektonické doporučení

Po verzi **0.5.5** bych přestal přidávat novou infrastrukturu a soustředil se na **doménové funkcionality**. Základ (workspace, projekty, pipeline, validace, registry, výsledky) bude dostatečně robustní a další vývoj by měl směřovat především k biologickým analýzám a interaktivní exploraci TCR repertoárů, což je hlavní přidaná hodnota celého projektu.
