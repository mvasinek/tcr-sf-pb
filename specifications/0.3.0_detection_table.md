Definice
sf_cells = počet buněk klonu v SF
blood_cells = počet buněk klonu v blood
detected_in_sf = sf_cells > 0
detected_in_blood = blood_cells > 0
shared_clone = detected_in_sf and detected_in_blood

Frakce:

sf_fraction = sf_cells / celkový počet buněk v SF pro daného pacienta a cell_type
blood_fraction = blood_cells / celkový počet buněk v blood pro daného pacienta a cell_type

Pokud pro daného pacienta chybí SF nebo blood, frakce pro chybějící kompartment nastav na 0.

Doporučený postup:

Načti clone_counts.csv.
Normalizuj compartment.
Odstraň řádky s nepodporovaným kompartmentem.

sample_group + patient + cell_type + compartment

blood_fraction
Aplikuj min_cells:
max(sf_cells, blood_cells) >= min_cells

CHANGELOG

Doplň:

## [0.3.0] - Build paired SF/blood detection table

### Added
- Construction of paired SF/blood clone detection table.
- Support for SF-only, blood-only and shared clones.
- Compartment normalization.
- Per-patient and per-cell-type clone fractions.
- CLI for detection table generation.
- Optional filters for cell type, sample group and minimum clone size.
- Unit tests for paired detection table generation.
Kritéria dokončení

Feature je hotová, pokud:

lze načíst clone_counts.csv,
vznikne paired_detection_table.csv,
tabulka obsahuje sjednocení SF a blood klonů,
správně rozlišuje SF-only, blood-only a shared klony,
správně počítá absolutní počty buněk,
správně počítá relativní frakce,
fungují parametry --cell-type, --sample-group, --min-cells,
všechny testy projdou přes pytest,
projekt má verzi 0.3.0.
