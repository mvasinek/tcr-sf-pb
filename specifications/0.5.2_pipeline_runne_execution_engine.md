# Feature 0.5.2 – Pipeline Runner & Execution Engine

# Cíl

Rozšířit verzi **0.5.1** o první funkční analytickou vrstvu.

Od této verze se aplikace stává **integrovaným bioinformatickým pracovním prostředím**, které umožňuje:

* spravovat projekty,
* spravovat datasety,
* spouštět analytické kroky,
* sledovat jejich stav,
* zobrazovat logy,
* ukládat historii běhů,
* garantovat reprodukovatelnost analýz.

GUI stále neimplementuje analytickou logiku.

Veškeré výpočty probíhají přes knihovní API.

# Cache

Pokud výstupy existují a vstupy se nezměnily:

Runner nabídne:

```text
Reuse existing output

Recompute
```

Implementovat:

```python
needs_recompute()
```

zatím pouze podle existence výstupů.

Hashování vstupů bude doplněno později.

# Poznámka k budoucím verzím

Verze **0.5.2** vytváří univerzální execution engine.

Od verze **0.5.3** bude implementována **adapter framework**, díky kterému bude možné připojovat datasety z různých platforem (10x Genomics, BD Rhapsody, AIRR, custom formáty) a převádět je do jednotného interního modelu bez změny analytických modulů.
