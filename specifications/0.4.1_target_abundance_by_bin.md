CHANGELOG

Doplň:

## [0.4.1] - Target abundance by clone-size bin

### Added
- Target abundance summaries by source clone-size bin.
- Mean and median target clone counts for all and detected-only clones.
- Per-bin scatter plots of source vs target clone counts.
- Per-bin boxplots of target clone counts.
- Export of bin-level visualizations under `outputs/bins/`.
- GUI-ready helper functions for bin-level visualizations.
Kritéria dokončení

Feature je hotová, pokud:

vzniká target_abundance_by_bin_summary.csv,
lze zjistit nejen pravděpodobnost detekce, ale i typický počet buněk v druhém kompartmentu,
pro každý direction/bin vznikne scatter plot,
pro každý direction/bin vznikne boxplot,
výstupy jsou v outputs/bins/...,
funkce jsou volatelné samostatně z budoucího GUI,
všechny testy projdou přes pytest,
projekt má verzi 0.4.1.
