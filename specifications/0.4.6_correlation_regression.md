# Feature 0.4.6 – Correlation and regression: blood fraction → SF fraction

## Cíl

Navázat na verzi `0.4.5`.

Cílem je kvantifikovat vztah:

```text
blood_fraction → sf_fraction

tedy zjistit, jak dobře velikost klonu v krvi odpovídá velikosti téhož klonu v synovii.

Tato feature se zaměřuje na spojitou analýzu velikostí klonů, nikoliv na binární expanzi.

Transformace hodnot

Protože frakce obsahují nuly a mají silně šikmé rozdělení, počítej i log-transformované hodnoty.

Použij pseudo-hodnotu:

pseudocount = 1e-6

Vytvoř interně sloupce:

log_blood_fraction = log10(blood_fraction + pseudocount)
log_sf_fraction = log10(sf_fraction + pseudocount)

CLI

Přidej nový příkaz:

  --input ./outputs/paired_detection_table.csv \
  --output-dir ./outputs

Volitelné parametry:

--cell-type CD8
--scope all_clones|shared_clones|blood_detected|sf_detected
--pseudocount 1e-6
--no-plots

Pokud není --scope zadán, spočítej všechny scope.

CHANGELOG

Doplň:

## [0.4.6] - Correlation and regression

### Added
- Correlation analysis between blood and SF clone fractions.
- Log-transformed fraction analysis with pseudocount.
- Linear regression model `log_sf_fraction ~ log_blood_fraction`.
- Regression prediction and residual export.
- Scatter, regression fit and residual plots.
- CLI for correlation and regression analysis.
Kritéria dokončení

Feature je hotová, pokud:

vzniká outputs/correlation_regression/correlation_summary.csv,
vzniká outputs/correlation_regression/regression_summary.csv,
vzniká outputs/correlation_regression/regression_predictions.csv,
jsou spočítány korelace pro všechny scope,
je fitován model log_sf_fraction ~ log_blood_fraction,
funguje filtr --cell-type,
funguje parametr --scope,
funguje --pseudocount,
funguje režim --no-plots,
vznikají všechny čtyři PNG grafy, pokud není --no-plots,
všechny testy projdou přes pytest,
projekt má verzi 0.4.6.
