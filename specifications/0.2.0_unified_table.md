  --input ./outputs/combined_annotations.csv \
  --cell-output ./outputs/cell_receptors.csv \

CHANGELOG

Doplň:

## [0.2.0] - Build cell receptor and clone count tables

### Added
- Productive TCR contig filtering.
- Dominant TRA/TRB chain selection per cell.
- Construction of paired and single-chain clonotype keys.
- Export of `cell_receptors.csv`.
- Export of `clone_counts.csv`.
- Optional `--paired-only` mode for clone counting.
- Unit tests for clonotype table generation.
Kritéria dokončení

Feature je hotová, pokud:

lze načíst combined_annotations.csv,
vznikne cell_receptors.csv,
vznikne clone_counts.csv,
každá buňka má maximálně jeden TRA a jeden TRB řetězec,
clonotype_key je deterministický,
clone_counts.csv obsahuje počty klonů podle pacienta, kompartmentu a typu buněk,
funguje parametr --paired-only,
všechny testy projdou přes pytest,
projekt má verzi 0.2.0.
