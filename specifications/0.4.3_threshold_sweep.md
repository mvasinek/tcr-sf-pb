## Cíl

Rozšířit verzi `0.4.2`.

Cílem je zjistit, jak stabilní jsou závěry o konkordanci expanze mezi SF a blood při různých definicích expandovaného klonu.

expansion_threshold = 0.005

Důležitá interpretace

Do README přidej varování:

Při příliš vysokém thresholdu může být počet expandovaných klonů velmi malý.
Podmíněné pravděpodobnosti pak mohou být nestabilní.
Vždy interpretujte společně s n_sf_expanded, n_blood_expanded a n_expanded_both.

CHANGELOG

Doplň:

## [0.4.3] - Expansion threshold sweep

### Added
- Sweep of expansion concordance metrics across multiple thresholds.
- Default threshold grids for fraction, cell-count and quantile expansion definitions.
- Global threshold sweep summary.
- Patient-level threshold sweep summary.
- Threshold sweep plots for conditional probabilities, Jaccard index and expanded clone counts.
- CLI for threshold sweep analysis.
Kritéria dokončení

Feature je hotová, pokud:

vzniká outputs/threshold_sweep/threshold_sweep_summary.csv,
vzniká outputs/threshold_sweep/threshold_sweep_by_patient.csv,
lze spustit sweep pro metody fraction, cells, quantile,
funguje vlastní seznam thresholdů přes --thresholds,
funguje filtr --cell-type,
funguje --no-plots,
vznikají všechny tři PNG grafy, pokud není --no-plots,
všechny testy projdou přes pytest,
projekt má verzi 0.4.3.
