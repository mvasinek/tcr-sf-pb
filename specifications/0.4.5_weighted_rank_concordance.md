# Feature 0.4.5 – Weighted Spearman rank correlation

## Cíl

Rozšířit verzi `0.4.4`.

Běžná Spearmanova korelace dává stejnou váhu malým i velkým klonům. Biologicky nás ale více zajímají největší/expandované klony.

Cílem je proto doplnit:

```text
weighted_spearman_fraction
weighted_spearman_rank

kde větší klony mají vyšší váhu.

Definice vah

Podporuj minimálně tři váhovací strategie:

mean_fraction
max_fraction
source_blood_fraction
mean_fraction
weight = (blood_fraction + sf_fraction) / 2
max_fraction
weight = max(blood_fraction, sf_fraction)
source_blood_fraction
weight = blood_fraction

Výchozí strategie:

weight-method = max_fraction
Weighted Spearman

Implementuj weighted Spearman jako:

Pearson correlation between ranks, weighted by clone weight

Tedy:

weighted_spearman_fraction =
weighted_pearson(rank(blood_fraction), rank(sf_fraction), weight)

A pro již existující ranky:

weighted_spearman_rank =
weighted_pearson(blood_rank, sf_rank, weight)

Pozor: protože rank 1 = největší klon, korelace ranků má být pozitivní, pokud velké klony odpovídají velkým klonům.

README

Doplň sekci:

Version 0.4.5 – Weighted Spearman rank correlation

Vysvětli:

proč běžná Spearmanova korelace nestačí,
že weighted Spearman zvýrazňuje největší klony,
podporované váhovací metody,
biologickou interpretaci:
Vysoká weighted Spearman korelace znamená, že největší klony v krvi
odpovídají největším klonům v synovii.

### Added
- Weighted Spearman correlation for clone-size concordance.
- Multiple clone-weighting strategies.
- Weighted rank correlation summary.
- Weighted Spearman bar plot by cell type.

Kritéria dokončení

Feature je hotová, pokud:

vzniká weighted_rank_correlation_summary.csv,
podporuje váhování mean_fraction, max_fraction, source_blood_fraction,
počítá weighted_spearman_fraction,
počítá weighted_spearman_rank,
obsahuje globální řádek patient = ALL,
funguje filtr --cell-type,
funguje režim --no-plots,
vznikají grafy v outputs/weighted_rank_concordance/,
všechny testy projdou přes pytest,
projekt má verzi 0.4.5.
