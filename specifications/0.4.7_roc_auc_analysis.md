Hraniční případy

Pokud pro kombinaci cell_type + predictor platí:

n_positive == 0

nebo:

n_negative == 0

pak:

nastav auc = NaN,
nastav average_precision = NaN,
nevytvářej ROC/PR body pro danou kombinaci,
zapiš varování do konzole.

Pokud má prediktor nulovou varianci:

scores.nunique() < 2

pak nastav metriky na NaN.

## [0.4.7] - ROC/AUC prediction of synovial expansion

### Added
- ROC/AUC analysis for predicting SF-expanded clones from blood-derived features.
- Support for multiple blood-based predictors.
- Average precision and precision-recall curve outputs.
- Best threshold estimation using Youden J.
- Prediction score table.
- ROC, PR, AUC and score distribution plots.
- CLI for ROC/AUC analysis.
Kritéria dokončení

Feature je hotová, pokud:

vzniká outputs/roc_auc/roc_auc_summary.csv,
vzniká outputs/roc_auc/roc_curve_points.csv,
vzniká outputs/roc_auc/pr_curve_points.csv,
vzniká outputs/roc_auc/prediction_scores.csv,
lze predikovat sf_expanded z krevních prediktorů,
fungují prediktory blood_fraction, blood_cells, blood_percentile, blood_rank_inverse, log_blood_fraction,
fungují metody expanze fraction, cells, quantile,
funguje filtr --cell-type,
funguje --predictors,
funguje --pseudocount,
funguje režim --no-plots,
vznikají PNG grafy, pokud není --no-plots,
